# Aizen Theme

Custom editor themes with warm peach glow and soft purple haze.

## Themes

- **Aizen Dark** - Midnight coding sessions with warm peach glow and soft purple haze
- **Aizen Light** - Sunrise clarity with vibrant citrus accents and crisp meadow greens

## Installation

1. Open VS Code / Cursor
2. `Cmd+K Cmd+T` (or `Ctrl+K Ctrl+T`)
3. Search for "Aizen Dark" or "Aizen Light"
